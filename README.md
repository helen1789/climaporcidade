-- Clima por Cidade

Busca do clima de uma cidade utilizando APIs públicas.

-- Funcionalidades

- Campo para digitar a cidade;
- Conversão de cidade para latitude e longitude usando Geocoding;
- Consulta do clima usando Open-Meteo Forecast;

-- APIs utilizadas

Geocoding:
https://geocoding-api.open-meteo.com/v1/search

Forecast:
https://api.open-meteo.com/v1/forecast
